# Borrower Risk Dashboard

## Project Overview
This Power BI dashboard analyzes borrower data to classify lending risk using DAX measures and interactive visuals.

## Objectives
- Segment borrowers by risk level
- Analyze employment impact on risk
- Compare income bands against risk
- Study dependents vs borrower distribution

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Key Insights
- Medium-risk borrowers formed the largest segment
- Employed borrowers showed lower risk profiles
- Higher income borrowers were mostly low risk
- Dependents influenced borrower risk exposure
